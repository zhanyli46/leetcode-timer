Timer Mode

The countdown timer mode is default. Starting times for different difficulties can be adjusted below.

Stopwatch Mode

The stopwatch mode tracks the time spent when solving a problem. The maximum recorded time is 99:59:59 (359999 seconds).

Adjusting Time

Hover mouse on the time display and use mouse wheel to scroll up (more time) or down (less time) to adjust time.
